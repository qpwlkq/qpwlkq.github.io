我常用的快捷键

Author: Qin Peng Date: September 29, 2020  21:48:22 Category: 经验之谈

写在前面

百度他不香吗?

快捷键学会常用的就行, 快速提高使用舒适度和速度
尽可能脱离鼠标, 毕竟电脑最开始么有鼠标这玩意, 并且你是专业的 .

最后列举了几个常用的Windows快捷键. 有些快捷键我也不用, 不管
有的功能可能描述不太准确, 自己动手用过之后才能知道到底怎么用

想不想玩转computer?

想不想让人直呼内行?

想不想被大家顶礼膜拜?

那就学些快捷键吧!!!

1. 最常用

CTRL + F 查找
CTRL + R 替换
CTRL + C/V/X 复制/粘贴/剪切
CTRL + A 全选
CTRL + X 剪切

CTRL + Shift + C/X 注释一行或者选中行/取消注释
CTRL + 滚轮(鼠标中间) 放大缩小
CTRL + D 向下复制一行

ALT + ↑/↓ 上移/下移代码
CTRL + L 剪切, 但是我把它当作删除一行使用...................滑稽

上面的快捷键几乎对于所有的软件通用, 和Windows系统

2. 关于编译器

2.1 code::blocks:

CTRL + F9 编译
右键可以格式化代码
太白了, 虽然可以更换主题, 但是主程序上面的工具栏无法更改, 好用是好用, 晚上对眼不好

ctrl + C 终止程序(开启编译后再黑色运行框框中)
ctrl + Z 终止输入

2.2 Visual studios :

Shift + F5 编译
ctrl + A 全选之后, 按下Ctrl + K , 松开, 再按Ctrl + F, 格式化代码!!!

黑色主题比较好看, 还算比较好用, 总体比较大, 写工程用的, 其实有点小题大做
时候的时候, 要关掉安全开发周期
注意不是标准的C++版本!!!
常常与CB输出不一样.
微软nb呗

CLion, Dev, …

不知道
其他编译器不管

4. Windows 10系统快捷键

强烈推荐!!! :

Win + ←/→/↑/↓ 能够分屏!
Win + I 设置
ALT + TAB 切换当前窗口
Win + X 开始菜单
Win + D 最小化所有窗口, 返回桌面
Win + L 锁屏防偷窥
Win + E 文件资源管理器
Win + T/数字键 选择任务栏
CTRL + Shift + Esc 任务管理器
ALT + F4 关闭
windows + W 截屏

5. 浏览器快捷键, 浏览器太多了, 自己百度吧(同等重要)
