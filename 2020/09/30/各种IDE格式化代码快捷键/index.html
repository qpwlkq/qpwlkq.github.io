各种IDE格式化代码快捷键

Author: Qin Peng Date: September 30, 2020  19:44:42 Category: 经验之谈

汇总一下各种IDE如何格式化代码

请友善对待各位师哥/师姐/队友吧!

不是不想给你讲, 奈何您的代码一坨翔 (┬┬﹏┬┬))

ಠ_ಠ

1.Codeblocks

鼠标点击右键, 点击”Format use Astyle”, 即可

CB没有自动设置快捷键, 所以要自己设置一下: 

第一步, 点击最上方倒数第二个Settings选项,
第二步, 下拉菜单中选择正数第二个Editor.
左边有一个Keyboard shortcuts选项, 点击
第三步, Commands窗口中点击Plugins选项, 下拉菜单中有一个"Source code formatter(Astyle)",点击
最后, 从右边输入快捷键.

2.Clion

好说好说
Ctrl + Alt + I 自动缩进
Ctrl + Alt + L 格式化

3.Visual Studio

有点点复杂
Ctrl + A全选代码
先同时按Ctrl + K
再同时按Ctrl + F

4.IDEA

Ctrl + Alt + L

5.Eclipse

Ctrl + Shift + F

其他的先不管
