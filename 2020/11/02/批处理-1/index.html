常用批处理

Author: Qin Peng Date: November 2, 2020  19:49:40 Category: 经验之谈

前言

XXXX 代表输入字符串
N 一个N代表输入一位16进制数字(多数字对应多个N)
NUM 代表输入一个数字

  1. title XXXX: 改变控制台标题(有个屁用???)

  2. color NN : 改变控制台的颜色, 两个16进制数字, 前一个背景色, 后一个字体颜色

  3. mode NUM: 改变控制台窗口大小

  4. cd :切换目录 {

    1. cd / : 切换到根目录
    2. cd .. : 切换到上一目录
    3. cd ../.. : 切换到上一个的上一个目录, 以此类推
    4. cd XXXX : 直接跳到XXXX目录
      }

  5. dir : 罗列目录下文件{

    1. dir /? : dir具体使用
    2. dir : 展示所有文件
    3. dir /ad : 显示文件夹(包括隐藏的)
    4. dir /a-d : 显示文件
      (包括隐藏的)
    5. dir /ah : 显示隐藏文件
    6. 很多不常用的无需学习
      }

  6. cls : 清屏

  7. md XXXX: 创建文件夹, 无法创建文件

  8. rd /s /q XXXX: 删除文件夹及其目录下所有内容, 如果不加/s, 目录不为空时无法删除.(/q 是不再二次确认, quiet安静)

  9. del XXXX: 删除文件(夹?), 删除文件不会进入回收站, 谨慎使用

  10. copy

  11. xcopy

  12. rename
