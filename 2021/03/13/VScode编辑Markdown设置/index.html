VScode编辑Markdown设置

Author: Qin Peng Date: March 13, 2021  21:27:27 Category: 经验之谈

###　前言

调教了一晚上vscode, 终于能够导出让我满意的pdf了
事情的起因是这样的: 我企图用md格式做一张单词表
打完之后发现, 未知原因单词压根对不齐, 心态崩了
经过和lhr的多次尝试, 偶然发现vscode上是等宽字体, 而预览的markdown不是等宽字体!!!
继而发现输出的pdf和预览也不一样, 问题发现了, 就是解决

安装插件：Markdown Preview Enhanced

直接vscode插件库中搜索点击安装即可

只使用默认设置也不错，但是不等宽字体会令人很难受

设置等宽字体要用css, 如果不会那就直接复制粘贴我写的一点点
ctrl + shift + P打开指令窗口
输入 :　Markdown Preview Enhanced: Customize CSS
即可打开css, 复制下面的粘贴过去覆盖掉原来的东西就行了
(大佬可以自己改样式)

不用自己设置的样式, 也可以选择它提供的一些样式, 个人感觉很鸡肋, 不会有程序员不用等宽字体吧

点击插件下面的小齿轮, 下拉菜单选择扩展设置
然后从上面的搜索栏中搜索theme(自带的一些命令不要删, 直接跟在后面加空格写theme), 有几个选项可以自己设置一下.

后记

借着这次机会好好看了看插件的doc, 确实很多功能我都没有用到, 自己看看文档再用也是不错的

https://shd101wyy.github.io/markdown-preview-enhanced/#/zh-cn/
