软件提高生产力的设置-必改

Author: Qin Peng Date: July 22, 2021  10:00:37 Category: 经验之谈

前言

说实话, 生产力工具好不好真的很难说, 类似VS, idea这一类比较大的软件都有相当多的复杂设置, 对于初次使用确实不太友好, 并且现有文章会告诉你如何修改它们的配置, 以提高使用舒适度.

我平时使用idea, 也是傻傻直接用, 甚至也算用过一些时间, 但从没改过什么, 直到前几天我看B站上黑马出的一套HarmonyOS的开发视频, 才知道原来还能这样(比如取消代码提示的大小写敏感, 这样敲个string就能出来String了, 巨舒服).

HarmonyOS开发工具DevEco是从idea的基础上开发的, 我说怎么这么眼熟呢.

idea

  1. 取消代码提示的大小写敏感:
    File -> Settings -> Editor -> General -> Code Completion -> Match case取消选择
