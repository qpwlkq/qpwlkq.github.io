zookeeper常用命令

Author: Qin Peng Date: August 2, 2021  0:00:00 Category: zookeeper

常用服务端命令

  • ./zkServer.sh start: 启动
  • ./zkServer.sh status: 查看状态
  • ./zkServer.sh stop: 停止
  • ./zkServer.sh restart: 重启

常用客户端命令

  • ./zkCli.sh -server IP:PORT: 指定连接的ip端口号.

  • ls /...: 查看某个节点下的子节点, 必须要加’/‘

  • create [-e/-s/-es] /... [可选-数据]: 在目录…下创建节点

  • get /...: 获取某个节点的数据, 空返回null

  • set /... data: 为某个节点设置节点

  • delete /...: 删除某个无子节点的节点

  • deleteall /...: 删除某个节点以及其所有子节点

  • help: 帮助

  • quit: 退出

  • ls -s /...: 查看某个节点的各种信息, 老版本是ls2命令
